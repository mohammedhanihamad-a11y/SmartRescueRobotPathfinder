# Project Description

Smart Rescue Robot Pathfinder is a JavaFX application that simulates an autonomous rescue robot operating inside a 10×10 grid environment.

The robot searches for victims, avoids obstacles, and finds the shortest path using the Breadth-First Search (BFS) algorithm. The project demonstrates Object-Oriented Programming (OOP), JavaFX GUI development, 2D arrays, searching algorithms, and autonomous decision making.

# Algorithms

### Breadth-First Search (BFS)

The BFS algorithm is used to calculate the shortest available path between the robot and the nearest victim while avoiding obstacles.
